# x86 Arch Directory Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Move the current x86-specific sources from `kernel/arch/` into `kernel/arch/x86/` without changing x86 runtime behavior.

**Architecture:** Keep this as a mechanical directory refactor. Move only the existing x86 architecture sources and headers, update the x86 object list and include path, then fix the small set of docs that still describe the old location.

**Tech Stack:** Freestanding C and NASM x86 kernel, GNU Make, QEMU-based x86 verification, repo docs under `docs/`.
